## How to Find More Player IDs

### Method 1: Liquipedia
1. Go to https://liquipedia.net/dota2/Portal:Players
2. Click on a player
3. Look for "Steam ID" or "Links" section
4. Steam ID is the number in the profile URL

### Method 2: Dotabuff
1. Go to https://www.dotabuff.com/
2. Search for player
3. Steam ID is in the URL: `dotabuff.com/players/XXXXXXXX`

### Method 3: Stratz
1. Go to https://stratz.com/
2. Search for player
3. Steam ID is in the URL

### Method 4: From In-Game
If you watch pro games in Dota 2 client, you can see player profiles.
